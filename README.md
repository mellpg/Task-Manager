Aplicação web desenvolvida para gerenciamento eficiente de tarefas cotidianas. Este projeto passou por um processo de refatoração estrutural com o objetivo de aplicar boas práticas de arquitetura de front-end, design responsivo e semântica, consolidados durante o curso de Desenvolvimento Web no **SENAC**.

---

## 🎯 Objetivos do Projeto e Refatoração

O foco principal desta atualização não foi apenas o aspeto visual, mas sim a entrega de um código limpo, otimizado e aderente aos padrões de mercado. A arquitetura foi redesenhada para garantir que a aplicação seja leve, acessível e escalável.

## 🚀 Implementações Técnicas

Durante a evolução do projeto, foram aplicados os seguintes conceitos fundamentais:

* **Abordagem Mobile-First:** A interface foi projetada e codificada tendo como prioridade os dispositivos móveis. Essa estratégia garantiu um consumo otimizado de recursos e uma experiência de uso fluida em ecrãs menores, expandindo-se progressivamente para desktop.
* **Grid System do Bootstrap (Regra das 12 Colunas):** Utilização rigorosa do sistema de colunas do framework para a distribuição dos elementos. Isso eliminou o uso de posicionamentos manuais complexos (*gambiarra*) e garantiu um alinhamento matemático e consistente dos componentes.
* **Responsividade Avançada:** Adaptação dinâmica da aplicação através de *breakpoints* estratégicos, garantindo que o gerenciador seja totalmente operacional em smartphones, tablets e monitores de alta resolução.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do projeto.
* **CSS3:** Customizações específicas e refinamento estético.
* **Bootstrap:** Framework utilizado para agilizar o desenvolvimento do ecossistema responsivo e componentes de interface.

---

## 📐 Padrão de Arquitetura Visual (Grid)

A estrutura do painel principal foi dividida utilizando a lógica flexível de colunas:
* **Ecrãs Pequenos (Mobile):** Elementos ocupam a largura total (`col-12`) para favorecer o toque e a leitura.
* **Ecrãs Médios/Grandes (Desktop):** Distribuição proporcional (ex: `col-md-4` para filtros/estatísticas e `col-md-8` para a listagem principal de tarefas).
